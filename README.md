# azure-cloud-course-itmt1378
This course introduces cloud computing, database management, and real-time analytics using Microsoft Azure. Students deploy virtual machines, design relational and NoSQL databases, and build data solutions with Microsoft Fabric through hands-on labs.

## Course Modules

1. Azure Virtual Machines
2. Azure Cosmos DB
3. Microsoft Fabric Lakehouse
4. Eventstream & Real-Time Analytics

## Learning Objectives
- Deploy cloud infrastructure
- Work with NoSQL databases
- Analyze data using Fabric
- Build real-time pipelines

## Tools Used
- Azure Portal
- Cosmos DB Emulator
- Microsoft Fabric
